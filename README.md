## 🛠️ طريقة الرفع على GitHub

### الخطوة 1 — إنشاء مستودع جديد
1. افتح [github.com](https://github.com)
2. اضغط **New repository**
3. اسم المستودع: `mohammed1`
4. اختر **Public**
5. لا تضع ✓ على "Add README" (موجود بالفعل)
6. اضغط **Create repository**

### الخطوة 2 — رفع الملفات
```bash
# داخل مجلد المشروع
git init
git add .
git commit -m "🚀 Initial commit - Mohammed1 Platform"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/mohammed1.git
git push -u origin main
```

### الخطوة 3 — تفعيل GitHub Pages
1. اذهب إلى **Settings** في مستودعك
2. من القائمة الجانبية اختر **Pages**
3. تحت Source اختر **Deploy from a branch**
4. اختر فرع `main` والمجلد `/ (root)`
5. اضغط **Save**
6. انتظر دقيقتين ثم افتح: `https://YOUR_USERNAME.github.io/mohammed1`
